dpuAgent Documentation

This repository currently hosts one Ploinky agent: dpuAgent.

The agent is a confidential data plane consumed by Explorer flows. It exposes MCP tools for secrets, confidential objects, comments, and ACL enforcement.

Tool execution is routed through dpuAgent/tools/dpu_tool.sh and dispatched by dpu_tool.mjs.

Operational Role

dpuAgent mediates between Explorer confidential workflows and encrypted DPU storage operations, with identity-aware access checks applied before protected values or content are materialized.

The same boundary separates metadata, ACL state, encrypted secret values, and confidential blob content, and it supports both Ploinky runtime execution and standalone MCP mode.

Repository Scope

The repository can host multiple agents over time. Current operational scope is centered on dpuAgent as the confidential runtime intermediary between Explorer and secure data storage.

Validation rule: run npm test from dpuAgent/ for code changes, together with documentation updates.
